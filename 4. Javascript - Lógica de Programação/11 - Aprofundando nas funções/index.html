Se aprofundando nas funções

Neste exercício vamos simular o computador de bordo de uma espaçonave no momento em que ela incia a parada até abrir as comportas para os pilotos saírem.

  1. Considerem que a nave está a 150km/s
  2. Crie uma função para desacelerar 20km/s a cada segundo. considere, no código, que cada iteração de um laço de repetição é 1 segundo.
  3. Esta função serã uma HOF e também deve imprimir cada atualização que houver na velocidade da nave. Esta impressão deve ser feita por meio de callback que essa função receberá
  4. Ao final, exiba uma mensagem informando que a nave está parada e as comportas podem ser abertas.
